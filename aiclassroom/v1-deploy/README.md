# 《父亲、树林和鸟》教学展示网站 - V1版本

## 项目概述

这是一个专门为三年级语文课文《父亲、树林和鸟》设计的教学展示网站。网站采用温暖的自然色调和优雅的排版设计，全面展示了这堂40分钟精读课的教学设计、实施过程和分析评估。

## 设计特色

### 视觉设计
- **配色方案**: 温暖的土色调（米色背景 #F8F6F0，深森林绿 #2D5016，中等绿色 #5A7C47，温暖金色 #D4A574）
- **字体选择**: Sorts Mill Goudy（标题）+ Oranienbaum（正文）
- **设计风格**: 现代教育美学，融合人文气息和学术氛围

### 交互体验
- **流畅动画**: 使用Anime.js实现页面元素的优雅动画效果
- **响应式设计**: 基于Tailwind CSS，适配各种设备屏幕
- **数据可视化**: ECharts.js展示课堂提问统计和分析

## 网站结构

### 1. 主页 (index.html)
- **Hero区域**: 展示课程主题和核心概念
- **课程亮点**: 6个核心知识点卡片展示
- **教学特色**: 教学设计创新点介绍
- **悬念设置**: "父亲到底是不是猎人？"互动环节

### 2. 教学流程页面 (process.html)
- **时间轴导航**: 40分钟课程的交互式时间轴
- **环节详解**: 10个教学环节的详细说明
- **进度可视化**: 实时显示教学进度
- **流程图解**: 教学流程的可视化展示

### 3. 课堂分析页面 (analysis.html)
- **数据统计**: 33次提问的统计分析
- **图表展示**: 提问类型分布和认知层次分析
- **效果评估**: 教学优缺点的详细分析
- **改进建议**: 具体的教学改进措施

## 技术实现

### 前端技术栈
- **HTML5**: 语义化结构
- **CSS3**: 现代样式和动画
- **JavaScript**: 交互功能和动画控制
- **Tailwind CSS**: 响应式布局框架

### 第三方库
- **Anime.js**: 轻量级动画库
- **ECharts.js**: 数据可视化图表
- **Google Fonts**: 优质字体资源

## 使用说明

### 本地运行
1. 解压压缩包到本地目录
2. 使用HTTP服务器运行（如 `python -m http.server 8000`）
3. 在浏览器中访问 `http://localhost:8000`

### 文件说明
- `index.html`: 网站主页，展示课程概览
- `process.html`: 教学流程页面，展示详细的教学过程
