# 语文教师教学评估报告可视化网页需求文档

## 1. 网站概述

### 1.1 网站名称
语文教师教学综合评估平台

### 1.2 网站描述
专业的语文教师教学评估报告展示平台，将复杂的教学评估数据进行可视化呈现，帮助教师直观了解教学表现，促进教学水平提升。

## 2. 核心功能

### 2.1 评估报告展示
- 教师基本信息：被评估教师姓名、授课内容、评估日期、授课班级等
- 时间统计：观察时间段和总时长显示
- 四维评估量表：教学目标与设计、教学互动与过程、阅读素养培育、教师专业素养四个维度的权重和评估结果
- 时间线分析：教学过程时间控制总体评价及改进建议

### 2.2 提问分析模块
- 布鲁姆目标分类学框架下的提问设计分析
- 优势展示：问题设计梯度、思维发展支持等
- 改进建议：高阶问题比例提升、讨论时间优化等
\n### 2.3 知识点总结
- 三个维度的知识点分类展示：语言知识与技能、阅读策略与方法、人文内涵与思维发展
- 具体知识点明细：生字学习、词语理解、写字技巧、朗读训练等内容
- PISA阅读素养体现：信息获取、文本理解、元认知策略等

### 2.4 对比分析
- 授课现场与教学设计的分析对比
- 一致性评价和优化建议展示

### 2.5 综合评分与反馈
- 突出优势展示：目标聚焦、互动效率、专业指导等方面
- 具体改进建议：高阶思维训练深化、环节时间优化等
- 总体评语可视化呈现

### 2.6 师生互动行为分析报告
- **报告摘要展示**：基于弗兰德斯互动分析系统（FIAS）的量化与质性分析结论，呈现整体互动特征和改进建议
- **整体互动结构与节奏分析**：
  - 弗兰德斯互动分析概览：师生话语比例、师生话语比（TSR）、互动模式等数据展示
  - 课堂节奏分析：不同时间段的互动特征、学生行为指标变化趋势
- **师生互动深度诊断**：
  - 教师主导下的高效互动分析
  - 学生参与的质量与局限评估
- **综合结论与教学建议**：
  - 总体评价展示
  - 精准教学改进建议列表
- **编制依据说明**：报告的数据来源和编制标准说明

## 3. 页面结构
\n### 3.1 报告首页
- 教师基本信息区域
- 四维评估量表概览
- 时间统计图表
- 师生互动行为分析报告入口\n
### 3.2 详细分析页
- 教学过程时间线分析
- 提问设计分析
- 课程知识点总结
- 对比分析结果
- 师生互动行为分析详细报告

### 3.3 反馈建议页
- 突出优势展示
- 具体改进建议
- 总体评语

## 4. 数据展示方式

### 4.1 图表展示
- 饼状图：四维评估量表权重分布
- 柱状图：时间分配统计
- 雷达图：布鲁姆目标分类学问题分布
- 柱状对比图：师生话语比例展示
- 折线图：课堂节奏波浪式演进趋势
- 表格：不同时间段的学生行为指标对比

### 4.2 文本展示
- 重点内容加粗显示
- 列表式知识点罗列
- 分层段落结构便于阅读
- 互动分析报告采用结构化分段展示

## 5. 设计风格

### 5.1 配色方案
主色调采用教育蓝（#4A90E2）搭配清新绿（#7ED321），体现专业性和成长性

### 5.2 视觉细节
采用卡片式布局，圆角设计增加亲和力，适度阴影提升层次感

### 5.3 整体布局
左侧导航栏配合主内容区域，响应式设计适配不同屏幕尺寸
