# 图像分类训练应用 - 设计文档

## 设计理念

### 视觉语言
- **科技现代主义**: 采用简洁的几何形状和精确的对齐，体现AI技术的精确性
- **深色主题**: 使用深色背景减少眼部疲劳，突出内容区域
- **智能色彩**: 使用低饱和度的专业色彩，避免视觉干扰
- **数据驱动美学**: 界面元素围绕数据和功能组织，而非装饰性元素

### 颜色方案
- **主背景**: `#1a1a1a` (深灰黑)
- **次背景**: `#2d2d2d` (中灰)
- **卡片背景**: `#3a3a3a` (浅灰)
- **主要文字**: `#ffffff` (纯白)
- **次要文字**: `#b0b0b0` (浅灰)
- **强调色**: `#4a9eff` (蓝色) - 用于按钮和重要信息
- **成功色**: `#4caf50` (绿色) - 训练成功、正确预测
- **警告色**: `#ff9800` (橙色) - 训练警告、低置信度
- **错误色**: `#f44336` (红色) - 错误状态

### 排版系统
- **主标题**: Noto Serif SC, 32px, font-weight: 700
- **副标题**: Noto Sans SC, 24px, font-weight: 600  
- **正文**: Noto Sans SC, 16px, font-weight: 400
- **小字**: Noto Sans SC, 14px, font-weight: 400
- **代码/数据**: 'Courier New', monospace, 14px

## 视觉效果

### 使用的库和效果
1. **Anime.js**: 用于按钮点击动画、卡片出现动画
2. **ECharts.js**: 训练过程的数据可视化图表
3. **p5.js**: 摄像头预览和图像处理效果
4. **Shader-park**: 背景动态效果
5. **Splide.js**: 样本图片轮播展示
6. **Matter.js**: 训练完成后的粒子庆祝效果
7. **PIXI.js**: 高性能图像渲染和特效

### 动画设计
- **微交互**: 按钮悬停时的轻微缩放(1.05x)和阴影增强
- **页面切换**: 淡入淡出过渡，持续时间300ms
- **数据更新**: 数字变化的计数动画
- **训练进度**: 平滑的进度条填充动画
- **结果展示**: 置信度条从左到右的填充动画

### Hero区域效果
- **动态背景**: 使用shader-park创建流动的几何图案
- **标题效果**: 渐变文本颜色循环，营造科技感
- **粒子系统**: 背景中缓慢移动的光点，象征数据流

### 交互反馈
- **按钮状态**: 
  - 默认: 半透明背景 + 细边框
  - 悬停: 背景变亮 + 轻微阴影
  - 激活: 缩放动画 + 颜色变化
- **卡片悬停**: 轻微上浮效果 + 阴影增强
- **图像上传**: 拖拽时的视觉反馈，边框高亮
- **训练状态**: 脉动动画表示正在进行

### 数据可视化风格
- **图表配色**: 使用单色系渐变，避免高对比度
- **线条样式**: 细线条(2px)，圆角处理
- **数据点**: 小圆点(4px)，悬停时放大
- **网格线**: 极淡的灰色，不干扰数据展示

### 响应式设计
- **断点设置**: 
  - 移动端: < 768px
  - 平板: 768px - 1024px  
  - 桌面: > 1024px
- **布局适配**: 移动端单列布局，桌面端多列布局
- **字体缩放**: 移动端字体适当缩小，保持可读性

### 特殊效果
- **加载状态**: 骨架屏 + 脉冲动画
- **成功反馈**: 绿色勾选动画 + 轻微震动
- **错误提示**: 红色摇摆动画 + 错误图标
- **训练完成**: 粒子爆炸效果 + 成功音效
