# 声音分类训练应用 - 交互设计

## 核心交互流程

### 1. 分类管理交互
- **分类创建**: 用户点击"添加分类"按钮，系统自动生成新分类（分类3、分类4等）
- **分类编辑**: 点击分类名称右侧的编辑图标，名称变为可编辑状态，按Enter或失焦保存
- **分类操作**: 点击三个竖点图标显示操作菜单（清空样本、删除类别）
- **样本管理**: 每个分类显示样本数量，最少需要1个样本才能训练

### 2. 音频录制交互
- **录制启动**: 点击麦克风按钮，切换到录制组件界面
- **录制过程**: 显示2秒倒计时进度条，实时录制音频
- **录制完成**: 自动保存音频样本，更新样本计数，返回麦克风按钮状态
- **录制取消**: 点击X按钮取消录制，返回初始状态

### 3. 模型训练交互
- **训练启动**: 点击"开始训练"按钮，显示训练进度条
- **训练过程**: 实时显示训练进度百分比和状态信息
- **训练完成**: 启用模型使用区域和下载按钮

### 4. 模型使用交互
- **实时检测**: 训练完成后显示"使用训练模型"按钮
- **音频检测**: 开启麦克风后实时分析音频，显示分类结果和置信度
- **结果展示**: 以进度条形式显示各分类的置信度百分比
- **麦克风控制**: 通过开关按钮控制麦克风的启停

### 5. 模型管理交互
- **模型下载**: 训练完成后启用下载按钮，下载包含model.json和weights的ZIP文件
- **状态提示**: 未训练时显示"至少训练一个模型"的提示信息

## 用户操作流程
1. 创建分类 → 2. 录制样本 → 3. 开始训练 → 4. 使用模型 → 5. 下载模型

## 交互特点
- 直观的视觉反馈：进度条、按钮状态变化、实时更新
- 流畅的操作体验：无需页面刷新，所有操作在单页面完成
- 智能的状态管理：按钮启用/禁用基于当前状态
- 响应式设计：适配不同屏幕尺寸
